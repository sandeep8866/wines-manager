📂 Google Drive PDF Folder
Save your ICDC PDF bills to a shared Google Drive folder, then paste the folder link here. Tap the button to fetch and process all new PDFs automatically.
How to set up:
1. In Google Drive, create a folder → name it "ICDC Bills"
2. No need to share publicly — the folder just needs to be in the same Google account as your Apps Script
3. Copy the folder link and paste above → Save Folder
4. Upload your ICDC PDF bills to that folder from any device
5. Fetch New Bills — only new PDFs are downloaded and processed
6. Re-fetch All Bills — re-downloads all PDFs to fix missing financial data
